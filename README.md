# Dino_Game

This repository contains "Dino Run", an endless-runner built with P5.js.

Key notes for agents and contributors:

- Requirements and gameplay spec: [PRD.md](PRD.md)
- Build output for P5 web editor: `BUILD/` (copy `sketch.js` into `BUILD/sketch.js` before pasting into https://editor.p5js.org/)

Avoid filenames with spaces or special characters when uploading assets to the P5 web editor.
